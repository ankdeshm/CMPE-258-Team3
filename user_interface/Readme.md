# Steps to run the UI:
1. Environment set up:

  ```python -m venv ui_env```
  
  ```source ./ui_env/bin/activate```

  ```pip install flask```
  ```pip install transformer```
  ```pip install sentencepiece```



2. Run app.py.py

  ```python app.py```


  
3. Open http://127.0.0.1:5000/ in a browser
